# Léo JEGO Portfolio

Portfolio personnel construit avec Next.js, React, TypeScript et MUI.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Contenu

Les contenus du profil et des projets sont centralisés dans `lib/locales`.

Le site expose aussi une version Markdown pour les agents via négociation HTTP :

```bash
curl -H "Accept: text/markdown" https://leo-jego.vercel.app/projets
```
